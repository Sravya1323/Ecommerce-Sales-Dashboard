# Ecommerce-Sales-Dashboard
An interactive Ecommerce Dashboard built in Microsoft Excel to analyze sales performance, product categories, customer trends, and merchant insights using dynamic charts, pivot tables, and data visualization techniques.
**Features**
Month-wise Sales Analysis
Top 10 Products by Category
Donut Charts for Category Distribution
Bar Charts for Comparative Insights
Amazon vs Merchant Performance Comparison
Geographic/Map-Based Sales Data
Interactive Dashboard with Visual KPIs

**Tools Used**
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Data Visualization Techniques

**Project Structure**
Dashboard → Main interactive dashboard
Month wise sales → Monthly sales trends
Top 10 products by category → Best-selling products
Amazon vs Merchant → Seller comparison
Charts → Donut & Bar chart analysis
Map data → Regional sales insights

**Objective
**To provide business insights and improve decision-making by transforming raw ecommerce data into meaningful visual analytics.
